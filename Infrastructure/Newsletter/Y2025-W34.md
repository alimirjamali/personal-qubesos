#### Epilogue
I have already upgraded a clone of `fedora-42-minimal` to Fedora 43 with the latest built packages. Upgrade of a clone of `fedora-42-xfce` template to Fedora 43 is under way at this moment. Official Fedora template in-place upgrade guide is almost sufficient for anyone who wants to give it a test. In case anyone sees error about `python3-gbulb`, just remove it from the template before in-place upgrade as it is no more needed by Qubes GUI tools. If you find any issue with the built packages, you could report them under [this issue](https://github.com/QubesOS/qubes-issues/issues/10102) or add a reference to it.

It should be also noted that XFCE4 WM is not available for Fedora 43 at this moment and using Fedora 43 as `sys-gui*` template with XFCE4 DE is not possible. There is ongoing debate and discussion between Fedora devs if X11 should be totally dropped in favour of Wayland for the final Fedora 43 release.
